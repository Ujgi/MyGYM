MyGYM Fitness Center Yönetim Sistemi

Bu projede ASP .NET CORE MVC teknolojisi kullanılarak MyGYM Fitness Center
isimli spor salonunun yönetim ve randevu sistemi oluşturulacaktır.

Projede üyeler istedikleri spor dalına göre ücret, antrenör yetkinlikleri ve özel dersler
için randevu alma imkanına sahio olacaklardır.

Antrenörler ile özel ders tercih etmeyen üyeler sisteme yükledikleri
boy, kilo, vücut tipi bilgisi ile yapay zeka aracılığıyle hedeflere göre
yapılması gereken antrenman programını ve diyet planı önerilerini alabileceklerdir.

Projede kullanılması hedeflenen teknolojiler

C#

Asp .Net Core MVC

SQL Server

Entity Framework Core

Bootstrap 5

HTML, CSS, JavaScript
